# samples/ — observer-facing blind set

Provenance-free captures, named by opaque sample-id only. Generated by
[`../tools/blind.py`](../tools/blind.py) from the sealed corpus. The provenance key lives in
`_sealed/manifest.csv`, which is gitignored and unsealed only at scoring.

Each sample is two PNGs: `S-NNNN-hero.png` (1280×800) and `S-NNNN-full.png` (1280×4200).

> **Do not judge this set and report a number.** It is a pilot of the blinding pipeline, not a
> runnable recognition set: it has no human side, so EXP-0004's primary hypothesis is undefined
> on it. Read [`../PILOT-CRITIQUE.md`](../PILOT-CRITIQUE.md) first.

If you are an EXP-0004 observer: these images are all you may see. Do not open `_sealed/`, the
manifest, or the git history of this directory before your answers are locked.
