## Layer 1 — Objective Observations

### Typography
- **Single typeface for the entire page: MSCHFSansMono** (custom monospace) — computed; no second font found in a 100-element sample.
- Uniform large list setting (computed base ~57.8px viewport-scaled on sampled links); all-caps project titles.
- Numbered index format: small "#nn" superscript-style numerals beside each entry (#1 … #88+).
- Case usage: ALL CAPS for all entries.

### Color
- **Two colors**: gray `rgb(132,137,142)` text on black `rgb(0,0,0)`; plus one chromatic element — "SHOP↗" with a yellow/red "!!!" chip.
- No other hue on the page.

### Depth & effects
- None. No gradients, shadows, blur, glow. 0 canvas elements, 0 videos, no visible imagery on the index.

### Shape & borders
- No buttons, cards, or radii; the only shaped element is the small rectangular "!!!" chip.

### Space & density
- A single continuous text list; uniform line rhythm; no sections, no whitespace bands.

### Layout & components
- Layout: one column of numbered text links (project drops, chronological), one floating "Shop↗" element top-right.
- Components: links. Nothing else — no nav bar, no footer detected in captures, no imagery.
- Content: ~88+ project entries ("JESUS SHOES", "TIMES NEWER ROMAN", "BIG RED BOOT"…).

### Motion
- Live page top renders content that the early headless frame missed — consistent with entrance animation or lazy render.

## Layer 2 — Relational Observations

_Observer-dependent; anchored to Layer-1 facts and screenshot `02` + live DOM._

### Visual hierarchy
Deliberately **flat to the point of absence**: one type size regime, one color, no imagery — no entry outranks another (anchor: single-font/single-color observations). The only hierarchical event on the page is the chromatic "SHOP↗ !!!", which marks commerce as the sole differentiated act.

### Refinement distribution
Uniform by subtraction: there is nothing to distribute refinement across. The refinement, if any, is in the typeface itself (custom mono) — the studio built a font to have no other design.

### Density rhythm
Constant: an unbroken list pulse from top to bottom. No alternation exists.

### Predictability
One schema (text list row), repeated ~88 times — structurally the most predictable page in the corpus, while being the least conventional page in the corpus. Predictability and convention-conformity fully dissociate here.

### Intentional asymmetry
The list is left-anchored with a single floating right-side element; no other compositional decisions are observable.

### Design risk
- Genre convention: studios/portfolios present work visually. Here **zero images** on the index of a company whose products are physical objects.
- Genre convention: navigation, footer, sections. **Absent.**
- Genre convention: multiple type styles. **One mono face, one size regime, one color.**
The page deviates from every observed genre convention simultaneously; the deviation set *is* the design.

### Perceived signature
The observer perceived the page as maximally attributable: gray mono all-caps numbered list on black + "!!!" chip. Any cropped fragment of three list rows would carry the attribution. Notably, the signature is produced **without any of the materials** (color, imagery, layout variation) that carry signature in every other corpus record so far.
