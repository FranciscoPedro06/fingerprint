# Experiments

Self-contained experiments that develop and stress-test the
[Observation Language](../observation-language/). Each experiment is a directory
`EXP-NNNN-‹slug›` with its own protocol, records, and artifacts.

## Convention

Each experiment MUST contain a `protocol.md` stating its purpose, method, and threats to
validity. Records produced by an experiment stay inside it until reviewed and promoted to
the [corpus](../corpus/).

## Current experiments

EXP-0001 and EXP-0002 matured the instrument. EXP-0003 onward serve the
[Research Program](../research/PROGRAM.md): building the corpus, measuring recognition,
attributing patterns, and testing reduction.

## How experiments feed the language

An experiment's instrument critique proposes changes; changes are ratified and folded into
the Observation Language or its profiles; the next iteration tests the result. The language
grows through this loop, not through ad-hoc field additions.
