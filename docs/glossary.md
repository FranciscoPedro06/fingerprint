# Glossary

Precise meanings of terms used across the project. Where a term has a normative definition
elsewhere, this glossary points to it rather than restating it.

- **Observer** — any agent capable of visual observation, human or machine. The grammar
  never depends on the observer's identity or internals.

- **Object** — the visual thing being observed, under one set of conditions. A different
  render, viewport, or state is a different object for recording purposes.

- **Evidence** — an individually verifiable statement about what is present in an object.
  Observer-independent target. (Observation Language, Specification §4.1.)

- **Relation** — a statement about how evidence units stand to one another within an
  object. Observer-relative but anchored. (Specification §4.2.)

- **Hypothesis** — a tentative statement that goes beyond the object, citing evidence and
  stating a refutation condition. (Specification §4.3.)

- **Attribution** — a ranked judgment of what an object appears to belong to or originate
  from, with declared confidence. (Specification §4.4.)

- **Attribution target** — what an attribution points to: a brand, author, studio,
  category, genre, design-system, production-method, era, medium, tradition, or
  `indeterminate`. (Controlled Vocabulary §6.)

- **Confidence** — an ordinal judgment of the strength of support within a record; not a
  probability. Levels: speculative, tentative, supported, strong. (Controlled Vocabulary §4.)

- **Immediacy** — how quickly an attribution formed: immediate, on-inspection, on-analysis.
  Records the "instant recognition" aspect observationally. (Controlled Vocabulary §7.)

- **Perceived conventionality** — a hypothesis that an object's recorded dimensions fall
  within the common conventions of its class. An observational construct, never a measured
  quantity.

- **Perceived proximity to genre conventions** — a hypothesis about how closely an object
  matches the norms of a named genre. Observational, not metric.

- **Profile** — an optional, additive catalog of evidence dimensions for a class of
  objects. Lives outside the core grammar and versions independently.
  (Observation Language, Profiles.)

- **Phenomenon** — a durable mechanism of recognition. (Methodology; Atlas.)

- **Fingerprint** — a perishable instance through which a phenomenon manifests at a point
  in time. (Methodology; Atlas.)

- **Provenance (of a record)** — whether the object's origin is known, unknown, or withheld
  to the observer. Unbiased attribution study requires it withheld. (Record Format §2.)

- **Corpus** — the collection of reviewed observation records. (Corpus.)

- **Atlas** — the catalog of phenomena and fingerprints, divided by status into
  experimental and validated. (Atlas.)
